# 首页导航一直高亮

## 现象

访问 `笔记`、`博客` 或 `关于` 页面时，顶部导航里的 `首页` 仍然保持高亮。

## 原因

首页路径是 `/`，其他页面路径也都以 `/` 开头。

Docusaurus 默认不是严格匹配导航路径，所以 `/docs/intro` 也会匹配到 `/`。

## 解决办法

给首页导航项加上 `exact: true`：

```ts
{to: '/', label: '首页', position: 'left', exact: true}
```

## 结论

如果导航项指向 `/`，通常应该加 `exact: true`，避免它在所有页面都高亮。
