# 笔记入口

这里是 **Kogorou Blog** 的笔记区。

它适合放更系统、更长期有效的内容，比如技术学习、项目实践、排错记录和读书摘要。

## 现在可以先这样用

- 遇到一个问题，就写到 `问题排查`。
- 学会一个概念，就写到 `技术笔记`。
- 做完一个小项目，就写到 `项目记录`。
- 暂时没想好分类，也可以先放在这里，之后再整理。

## 推荐写法

每篇笔记不用很长，先写清楚三件事就够了：

- 背景：为什么要记录这件事？
- 过程：做了什么，踩了什么坑？
- 结论：下次遇到同类问题该怎么做？

## 常用命令

本地预览：

```bash
npm run start
```

构建检查：

```bash
npm run build
```
